# Save The Date – [Thy & Caio]

Esta é a página oficial de Save The Date, desenvolvida para proporcionar uma experiência digital elegante e funcional aos convidados do nosso casamento. A página foi construída utilizando HTML, CSS e JavaScript, seguindo boas práticas de web design responsivo, garantindo compatibilidade em dispositivos móveis, tablets e desktops.

<b> • Funcionalidades </b>

- Design responsivo: layout adaptável para diferentes tamanhos de tela.
- Navegação intuitiva: seções organizadas para facilitar a visualização das informações do evento.
- Interatividade sutil: efeitos de transição e animações leves para aprimorar a experiência do usuário.
- Acessibilidade: uso de cores, contrastes e tipografia legíveis, pensando em todos os visitantes.

<b>  •  Estrutura do Projeto </b>

O projeto está estruturado em pastas claras, separando assets, estilos, scripts e imagens, facilitando manutenção futura e possíveis atualizações.
> Este repositório é uma representação digital do nosso convite, combinando estética minimalista e tecnologia web moderna, com foco em experiência do usuário e apresentação elegante.
